# ⚡ EV Secure Hub

## 🔐 Blockchain-Integrated EV Charging Platform

EV Secure Hub is a futuristic and secure Electric Vehicle charging management platform built using **Flask**, **Blockchain Technology**, **Encryption Protocols**, and **Secure Wallet Transactions**.

The platform simulates a secure EV charging ecosystem with:

* ⚡ Smart charging stations
* 🔐 Encrypted communication
* ⛓️ Immutable blockchain transaction ledger
* 💳 Secure wallet & payment handling
* 🧬 OTP-based authentication
* 📊 Real-time station monitoring dashboard
* 🌌 Cyberpunk-inspired UI/UX

---

# 🚀 Features

## 🔒 Security Features

* AES/Fernet-based data encryption
* Secure authentication system
* OTP verification mechanism
* Session-based authorization
* Blockchain-backed transaction integrity
* Tamper-resistant transaction storage
* Atomic wallet deduction system

---

## ⛓️ Blockchain Integration

Every EV charging transaction is stored inside a local blockchain ledger.

### Blockchain Features:

* SHA-256 hashing
* Immutable transaction history
* Genesis block initialization
* Chain validation
* Local blockchain storage using JSON

File Used:

```bash
blockchain_data.json
```

# 🛠️ Installation Guide

## 1️⃣ Clone the Repository

```bash
git clone <your-github-repository-link>
cd EV
```

---

# ⚙️ Setup Project

## Windows Users

Run:

```bash
setup.bat
```

This will:

* Create Python virtual environment
* Install required packages
* Initialize the database
* Configure dependencies

---

# ▶️ Run the Application

Run:

```bash
run_all.bat
```

Application will start at:

```bash
http://127.0.0.1:5000
```

# 📦 Manual Installation

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / Mac

```bash
source venv/bin/activate
```

## Install Dependencies

